# Hand Gesture Recognition — Himax WE-I Plus

Real-time hand gesture recognition using TinyML on the Himax WE-I Plus EVB.

## Gestures
- ✊ Fist
- 🖐️ Open Palm
- ✌️ Peace
- 👍 Thumbs Up
- 👈 Swipe Left
- 👉 Swipe Right
- 👋 Wave

## Stack
- TensorFlow Lite for Microcontrollers
- Himax WE-I Plus SDK
- Python/Keras (training)
- C/C++ (firmware)
